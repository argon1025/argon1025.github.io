MAC Homebrew 패키지 관리자를 통한 Mysql 설치와 세팅

2021-01-17 MySQL BabelFish, Project Comments Word Count: 403(words) Read Count: 2(minutes)

1.Mysql 설치

mysql을 설치하기전 패키지 매니저를 업데이트해야합니다

Homebrew 패키지 관리자에 등록된 mysql을 확인합니다 mysql@버전 으로 이전버전을 설치할 수 있습니다

mysql 설치를 진행합니다 버전을 입력하지 않았을경우 최신버전이 설치됩니다



2.Mysql 실행

mysql 서비스를 실행합니다



3.Mysql 설정



3-1.Mysql 설정_비밀번호 복잡성 설정

Y 복잡한 비밀번호 사용
N 간단한 비밀번호 사용



3-2.Mysql 설정_비밀번호 설정

root 계정 비밀번호를 설정합니다



3-3.Mysql 설정_사용자 설정

Y 익명사용자 접근 금지
N 익명사용자 접근 허용

Y 입력시 -u옵션을 사용해서 접속을 해야합니다



3-4Mysql 설정_root 원격 접속 허용 설정

Y root계정 원격 접속 불가
N root계정 원격 접속 가능



3-5.Mysql 설정_변경된 설정 적용여부 설정

Y 바로적용
N 적용하지 않음



4.Mysql 접속

mysql -uroot -p 명령어로 접속이 가능합니다
