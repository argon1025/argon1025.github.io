NodeJS에서 해시 알고리즘을 사용하기 위해서는 `crypto` 모듈을 사용하면 됩니다.

crypto 모듈은 callback을 지원하는 함수로 되어 있기때문에 서버에 적용하기전 async형태로 리팩토링 하였습니다

1.로직 작성

모듈 임포트


클래스 작성

Crypto 라는 클래스를 작성 하였습니다

crypto.pbkdf2

crypto.pbkdf2 는 패스워드, 솔트값, 암호화 연산 반복 횟수, 해시길이, 암호화 방식, 콜백함수를 인자로 받아 Hash 암호화한뒤 값을 derivedKey로 buffer를 반환합니다

이론편에서 언급한
서버는 패스워드를 Hash 암호화로 난독화 ( “1234” → “11161b2f1e” ) 부분입니다
바로가기

crypto.pbkdf2 리팩토링

crypto.pbkdf2 를 Promise 형태로 Crypto 클래스 내부 메서드로 리팩토링 하였습니다


crypto.randomBytes

crypto.randomBytes 해시길이, 콜백함수를 인자로 받아 랜덤한 salt buffer를 반환합니다.

이론편에서 언급한
서버에서 Salt값을 발급 (Salt : “34309e”) 부분입니다
바로가기


crypto.randomBytes 리팩토링

crypto.randomBytes 를 Promise 형태로 Crypto 클래스 내부 메서드로 리팩토링 하였습니다



hashencryption 메서드 추가

위에서 리팩토링한 메서드를 이용해서

비밀번호만 입력하면 알아서 해쉬값을 반환해주는 메서드를 추가했습니다

text:password(패스워드), int:keyLength = 64(해시 길이), int:iterationsCount = 100000(암호화 연산 반복횟수)

String:digSet = ‘sha512’(암호화 방식), text:salt(디코드할경우 기존 솔트값)

를 인자값으로 받고 salt값이 undefined인경우엔 새로 salt를 생성해서 암호화 하고 해시암호화한 Key와 사용한 Salt값을 반환합니다
